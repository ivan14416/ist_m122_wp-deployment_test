== Description ==

Istante is a clean minimal child theme of Avventura Lite. Fully optimized for the mobile devices, Istante is the perfect theme for your personal, travel, beauty, lifestyle, news or photography website. Istante has new features, like a sticky post grid on homepage, a featured links section, an optional header image and much more.

Developed by ThemeinProgress, https://www.themeinprogress.com
